## 项目主要需求

### 项目概述
**项目类型：Web前端**
搭建一个思维导图编辑器，用户可在线创建、编辑、导出思维导图，支持节点拖拽、快捷键操作和多种导出格式。

### 功能需求

1. **思维导图编辑**：中心节点向外扩展，支持添加子节点、兄弟节点，节点支持编辑文字、设置颜色标签、添加图标。
2. **节点操作**：拖拽移动节点位置、拖拽调整节点层级关系、折叠/展开子节点分支、删除节点及其子树。
3. **快捷键支持**：Enter 添加兄弟节点、Tab 添加子节点、Delete 删除节点、F2 编辑节点文字、Ctrl+Z 撤销、Ctrl+Y 重做。
4. **画布控制**：鼠标滚轮缩放、拖拽平移画布、适配全部（Fit to Screen）、居中选中节点。
5. **主题样式**：提供多种预设配色方案（商务蓝/活力橙/清新绿/极简黑白），支持自定义节点背景色和文字颜色。
6. **导出功能**：导出为 PNG 图片、JSON 数据文件（可重新导入编辑）、Markdown 大纲格式。
7. **自动保存**：编辑内容实时保存到 IndexedDB，支持多份思维导图文件管理（新建/重命名/删除/切换）。
8. **小地图导航**：右下角展示缩略小地图，点击可快速跳转到画布对应区域。

### 技术要求

- 前端框架：React 18 + TypeScript
- 画布引擎：reactflow（节点-连线图编辑框架）
- UI 库：Tailwind CSS + Radix UI
- 状态管理：Zustand + immer
- 构建工具：Vite
- 数据持久化：浏览器 IndexedDB
- 导出：html-to-image（PNG 导出）
- 代码规范：严格遵循单一职责原则，每个文件只负责一个功能；常量（快捷键映射、主题配置、节点类型）统一维护在 `src/constants/` 目录下；类型定义统一在 `src/types/`；所有工具函数封装在 `src/utils/`。

## 代码存放规则

这是纯前端单页应用（SPA）项目，无后端、无数据库。

前端代码统一放在 `frontend/`：

```
frontend/
├── src/
│   ├── pages/              # 页面组件
│   ├── components/
│   │   ├── MindMap/        # 思维导图核心组件（节点、画布、工具栏）
│   │   ├── Sidebar/        # 侧边栏文件管理
│   │   └── common/         # 通用组件
│   ├── hooks/              # 自定义 Hooks（useKeyboard, useAutoSave 等）
│   ├── stores/             # Zustand 状态管理
│   ├── storage/            # IndexedDB 封装
│   ├── constants/          # 常量定义
│   ├── types/              # TypeScript 类型
│   ├── utils/              # 工具函数（导出、序列化等）
│   ├── styles/             # 全局样式
│   └── App.tsx
├── public/
└── package.json
```
